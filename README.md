# CubeMart Checkout Service

![CubeMart](https://img.shields.io/badge/CubeMart-Checkout-0f766e)
![Go](https://img.shields.io/badge/Go-Service-00ADD8)
![gRPC](https://img.shields.io/badge/gRPC-Orchestration-1f2937)

Order orchestration microservice for CubeMart that coordinates cart, payment, shipping, currency, and email workflows during checkout.

Run the following command to restore dependencies to `vendor/` directory:

    dep ensure --vendor-only
